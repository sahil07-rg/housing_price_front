## ❈ Welcome ❈

</div>

➔ AI Housing Intelligence is a modern full-stack machine learning platform developed for intelligent property valuation and advanced housing analytics.

➔ The platform combines artificial intelligence, machine learning inference, analytics visualization, and responsive frontend engineering into a single integrated production-ready system.

➔ Inspired by luxury traditional aesthetics and cinematic interface design, the project delivers a premium AI-driven experience for analyzing property valuations using demographic and housing parameters.

➔ Whether you're exploring AI-powered valuation systems, modern frontend engineering, backend API integration, or machine learning deployment workflows, this repository provides a complete practical implementation of a production-style AI platform.

## ❈ Machine Learning Pipeline 🧠 ❈

</div>

### ✦ Input Parameters

The prediction model utilizes the following housing attributes:

- longitude
- latitude
- housing_median_age
- total_rooms
- total_bedrooms
- population
- households
- median_income
(Input=1.5 Means~$15,000)
- ocean_proximity
(<1H OCEAN,INLAND,ISLAND,NEAR BAY,NEAR OCEAN)

<br>

### ✦ Prediction Workflow

```text
User Input
      ↓
Frontend Validation
      ↓
FastAPI Backend API
      ↓
Machine Learning Inference
      ↓
Prediction Generation
      ↓
Analytics Visualization
      ↓
PDF Intelligence Report
```

## ❈ Local Setup 🚀 ❈

</div>

### ✦ Clone Repository

```bash
git clone https://github.com/sahil07-rg/housing_price_front.git
```
<br>

### ✦ Frontend Setup

```bash
npm install
npm run dev
```
<br>

### ✦ Backend Setup

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
